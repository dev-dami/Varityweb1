# Varityweb

Varityweb is a premium, fully-managed website builder engineered from the ground up for Nigeria. Build, design, and launch high-performance websites without touching code, managing complex servers, or worrying about hosting configurations.

## Introduction

Varityweb is a full-stack Next.js application offering multi-tenancy, custom domain support, and a highly customizable drag-and-drop web editor. It is optimized to deliver lightning-fast loading speeds on standard local networks and integrates seamlessly into the modern web ecosystem.

## Key Features

- **High-Performance Speed:** Local caching and lightweight pages optimized to load instantly on Nigerian mobile networks.
- **Worry-Free Infrastructure:** Automatic SSL certificates, managed hosting, database replication, and daily backups.
- **Client & Session Management:** Secure, lightweight authentication powered by Better Auth.
- **Drag-and-Drop Editor:** An intuitive interface allowing creators and businesses to publish websites effortlessly.
- **Multi-Tenancy & Subdomain Routing:** Dynamically rewrite subdomains (`app.varityweb.com`, `editor.varityweb.com`, or custom subdomains) out of the box.
- **Custom Domains:** Allow pro users to map their own domains (e.g. `company.com`) to their Varityweb sites.

## Tech Stack

This project is built with the following modern technologies:

- **Next.js (App Router)**: High-performance React framework.
- **Tailwind CSS**: Modern styling configuration.
- **Shadcn UI**: Highly interactive, responsive UI primitives.
- **Supabase PostgreSQL**: Highly scaleable database storage.
- **Better Auth**: Complete client-server session handling.
- **Prisma ORM**: Type-safe database client.
